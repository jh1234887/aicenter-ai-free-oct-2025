ChatGPT 무료 플랜

인기

OpenAI의 ChatGPT는 2025년 가장 강력한 무료 플랜을 제공합니다.

접근 가능한 모델

무료 플랜 사용자는 최신 GPT-5 모델에 제한적으로 접근할 수 있습니다. 5시간 동안 10~60개의 메시지를 보낼 수 있으며, 이 한도에 도달하면 GPT-4o mini와 같은 더 가벼운 모델로 자동 전환됩니다.

주요 기능

  • 웹 검색: 실시간 데이터를 검색하여 최신 정보를 제공합니다
  • 파일 업로드: 문서와 이미지를 업로드하여 분석할 수 있습니다(제한적)
  • 음성 모드: 표준 음성 대화 기능을 사용할 수 있습니다
  • 이미지 생성: DALL-E를 통해 하루 2-3개의 이미지를 생성할 수 있습니다
  • 캔버스: 문서 작성 및 코드 편집을 위한 인터랙티브 공간
  • GPT Store: 커스텀 GPT를 발견하고 사용할 수 있습니다
  • 프로젝트: 대화와 파일을 조직화할 수 있습니다

제한사항

무료 플랜의 가장 큰 제약은 프리미엄 모델에 대한 동적 사용 제한입니다. 서버 부하와 쿼리 복잡도에 따라 메시지 한도가 유동적으로 변합니다. Deep Research는 제한적으로만 제공되며, Tasks, 고급 음성 모드(비디오 포함), Sora 비디오 생성은 지원되지 않습니다.

학생 혜택

미국과 캐나다의 대학생은 .edu 이메일 인증을 통해 2개월 무료 ChatGPT Plus를 받을 수 있습니다.

Perplexity AI 무료 플랜

Perplexity는 실시간 검색과 인용에 특화된 AI 검색 엔진입니다. 2025년 10월 Comet brower를 모든 사용자에게 오픈하여 무료 사용자도 perplexity AI가 탑재된 브라우저를 경험할 수 있습니다.

검색 능력

무료 플랜은 무제한 Quick Search를 제공합니다. 이는 기본 모델을 사용하여 빠른 정보 검색과 정확한 답변을 제공합니다. 더 깊이 있는 분석이 필요한 경우, 하루 5회의 Pro Search를 사용할 수 있습니다. Pro Search는 GPT-4, Claude 3 Sonnet, Claude 3 Opus 등 고급 모델에 접근합니다.

컨텍스트 및 파일 처리

무료 플랜은 128,000 토큰의 컨텍스트 윈도우를 제공하며, 이미지는 최대 40MB까지 업로드할 수 있습니다.

Focus 모드

무료 사용자도 학술 논문, Reddit, YouTube, 뉴스 등 특정 소스에서 정보를 검색할 수 있는 Focus 모드를 사용할 수 있습니다.

제한사항

Deep Research, 이미지 생성, API 접근, 커스텀 모델 선택은 Pro 플랜($20/월)에서만 가능합니다. 무료 플랜에서는 답변 카드에 광고가 표시되며, 기본적으로 데이터가 향후 모델 훈련에 사용될 수 있습니다(옵트아웃 가능).

Google Gemini 무료 플랜

Google의 Gemini는 Google 서비스 생태계와 완벽하게 통합된 AI입니다.

모델 접근

무료 플랜은 Gemini 2.5 Flash 모델에 접근할 수 있습니다. 이는 표준 텍스트와 이미지 채팅, 검색 통합, 짧은 컨텍스트 추론을 제공합니다. 무료 사용자에게는 주기적으로 리셋되는 사용량 캡이 있습니다.

기본 기능

  • 멀티모달 기능: 기본 이미지 생성 및 컨텍스트 추론
  • Google 서비스 통합: Gmail, Drive, Docs 등과 연동
  • 웹, iOS, Android 접근: 모든 플랫폼에서 사용 가능

제한 기능

Deep Research, NotebookLM 통합, Google Drive grounding은 무료 플랜에서 비활성화됩니다.

학생 혜택

미국, 일본, 인도네시아, 한국, 브라질의 만 18세 이상 대학생은 15개월 무료 Google One AI Premium 플랜을 받을 수 있습니다. 이 플랜에는 Gemini Advanced, NotebookLM Plus, 2TB 스토리지, Deep Research, Gemini Live, Veo 2 비디오 생성 등이 포함됩니다.

Claude AI 무료 플랜

Anthropic의 Claude는 안전성과 긴 컨텍스트 처리에 강점을 가진 AI입니다.

사용 한도

무료 플랜은 하루 50개의 메시지를 보낼 수 있습니다. 사용량은 5시간 세션으로 관리되며, 각 세션이 끝나면 할당량이 리셋됩니다. 메시지 길이와 시스템 부하에 따라 실제 사용 가능한 메시지 수는 약 20~50개입니다.

모델 접근

무료 사용자는 Claude 3 Haiku에 접근할 수 있습니다. 이는 Claude 3 제품군 중 가장 빠르지만 능력이 제한적인 모델입니다. Claude 3.5 Sonnet이나 Claude 3 Opus와 같은 더 강력한 모델은 Pro 플랜($20/월)에서 사용 가능합니다.

기본 기능

  • 웹, iOS, Android 접근
  • 이미지 및 문서 분석: 거의 모든 문서와 이미지를 분석, 설명, 요약, 전사할 수 있습니다
  • 최신 모델 접근: Claude의 최신 모델 중 하나에 접근

제한사항

프로젝트(Projects), 지식 베이스, 조직 기능, 모델 선택권, API 접근, 우선 지원은 유료 플랜에서만 제공됩니다. 피크 시간대에는 무료 사용자의 응답이 느려지거나 일시적으로 사용할 수 없을 수 있습니다.

Microsoft Copilot 무료 플랜

Microsoft Copilot은 Windows, Edge 브라우저, Office 제품군과 통합된 AI 어시스턴트입니다.

접근 방법

무료 Copilot은 세 가지 방법으로 사용할 수 있습니다:

  • 웹 브라우저: copilot.microsoft.com 방문
  • 모바일 앱: iOS 및 Android 앱 다운로드
  • Edge 브라우저: 브라우저 우측 상단의 Copilot 아이콘 클릭

모델 및 기능

무료 플랜은 데스크톱에서 GPT-3.5를 사용하지만, 모바일에서는 GPT-4o-light에 무료로 접근할 수 있습니다. 사용자는 하루 15 boosts를 받아 이미지를 생성하고 비피크 시간에 최신 AI 모델에 접근할 수 있습니다.

  • 실시간 웹 검색 및 답변
  • AI 이미지 생성 및 편집
  • 멀티모달 기능: 텍스트, 음성, 이미지 처리

Pro 플랜

$20/월의 Copilot Pro는 향상된 추론 모델, 더 긴 컨텍스트 처리, 더 빠른 응답 시간을 제공합니다. iOS와 Android에서 30일 무료 체험이 가능합니다.

Le chat - 오픈소스(Mistral) 기반

Mistral AI의 chat 형식의 플랫폼입니다.

무료 플랜

Le Chat은 무료 플랜을 제공합니다.

무료 플랜 기능

  • 하루 25개 메시지 상당의 사용량
  • Mistral Small, Mistral Large, Mistral Next 모델 등 Mistral AI의 모델을 기반으로 선택하여 답변을 제공
  • 빠른 답변 기능인 'Flash Answers' 사용 가능(1,000단어/초)
  • 코드 인터프리터
  • 문서 업로드
  • 검증된 뉴스 검색

뤼튼(Wrtn) - 한국형 AI

한국

뤼튼은 한국 사용자를 위해 최적화된 생성형 AI 플랫폼입니다.

완전 무료 모델

2023년 12월 20일부터 뤼튼은 모든 기능을 전면 무료로 개방했습니다. 무료 플랜에서 무제한 사용이 가능하며, 광고 기반 수익 모델을 채택하고 있습니다.

다중 모델 접근

뤼튼 AI 서포터는 사용자의 질문에 맞춰 GPT-4o, GPT-4.5, OpenAI o1, Claude 3.7, Gemini 2.5 등의 모델을 자동으로 선택하여 호출합니다. 최신 자료가 필요한 경우 자동으로 인터넷 검색을 수행합니다.

기본 채팅이 아닌 GPT-5 채팅방에서는 GPT-5를 단독 사용 가능합니다.

NotebookLM - Google의 연구 도구

NotebookLM은 Google이 개발한 AI 기반 연구 및 학습 도구입니다.

무료 플랜

NotebookLM은 기본적으로 무료로 제공됩니다. 무료 플랜에서도 강력한 기능을 사용할 수 있습니다.

무료 플랜 제공 내용

  • 월 100개 노트북: 최대 100개의 노트북 생성
  • 노트북당 50개 소스: 각 노트북에 최대 50개의 소스 추가
  • 일 50개 쿼리: 하루 50개의 대화 쿼리
  • 일 3회 Audio Overview: 문서를 팟캐스트 스타일로 변환(하루 3회)
  • 소스당 50만 자: 각 소스는 최대 50만 자까지 지원
  • 노트북 공유: 제한적인 공유 기능

학생 혜택

미국, 일본, 인도네시아, 한국, 브라질의 대학생은 12개월 무료 Google AI Pro 플랜(NotebookLM Plus 포함)을 받을 수 있습니다.

GitHub Copilot - 개발자용 AI

개발자

GitHub Copilot은 Microsoft가 제공하는 AI 코드 어시스턴트입니다.

무료 플랜

2024년 12월부터 GitHub Copilot Free가 출시되었습니다.

무료 플랜 제공 내용

  • 월 2,000 코드 완성(completions)
  • 월 50 에이전트 모드 또는 채팅 요청
  • Claude 3.5 Sonnet, GPT-4.1 등 최신 모델 접근
  • 공개 코드와 일치하는 제안 차단

제한사항

무료 플랜은 VS Code에 통합되어 occasional 사용자에게 적합합니다. GitHub 계정이 필요합니다.

Pro 플랜 & 무료 혜택

$10/월(또는 연 $100)의 Pro 플랜은 GPT-5 mini를 사용한 무제한 에이전트 모드 및 채팅, 무제한 코드 완성, Claude 4, GPT-5, Gemini 2.5 Pro 등 추가 모델 접근을 제공합니다.

검증된 학생, 교사, 인기 오픈소스 프로젝트 유지관리자는 무료로 Pro 플랜을 사용할 수 있습니다.

                   

Google Gemini CLI - 개발자용

                  개발자                
               

Gemini CLI는 개발자가 터미널에서 직접 Google의 Gemini 모델을 활용할 수 있게 해주는 강력한 커맨드 라인 인터페이스입니다. 스크립팅 및 자동화에 특히 유용합니다.

                               
                   

무료 플랜 기능

                   

Gemini API의 무료 등급(Free Tier) 정책에 따라 CLI에서도 무료 사용이 가능합니다.

                   
                           
  • Gemini 2.5 Flash 모델 접근: 웹 UI와 동일한 무료 모델을 사용하여 텍스트 및 코드 생성이 가능합니다.
    •                        
  • 스크립팅 및 자동화: 셸 스크립트(Bash, Zsh 등)와 결합하여 반복적인 작업을 자동화할 수 있습니다.
    •                        
  • 스트리밍 출력: 모델이 생성하는 응답을 실시간으로 터미널에서 확인할 수 있습니다.
    •                        
  • 다양한 입력 처리: 직접 입력뿐만 아니라 파일이나 파이프(|)를 통해 다른 명령어의 출력을 입력으로 받아 처리할 수 있습니다.
    •                    
               
               
                   

제한사항

                   

무료 플랜은 API 사용량 제한(Rate Limit)이 적용됩니다. 현재 분당 60개의 요청(RPM)으로 제한되어 있어 대규모 배치 작업에는 한계가 있을 수 있습니다. 더 높은 처리량이 필요할 경우 유료 플랜으로 전환해야 합니다.

               
           

무료 사용 팁

1

효율적인 프롬프트 작성

무료 플랜의 메시지 제한을 고려하여, 한 번에 명확하고 구체적인 질문을 작성하세요.

2

여러 AI 병행 사용

각 AI의 강점을 활용하여 용도에 맞게 병행 사용하면 무료 플랜의 한계를 극복할 수 있습니다.

3

학생 혜택 활용

학생이라면 ChatGPT, Google Gemini, Microsoft Copilot의 무료 또는 할인 프로그램을 적극 활용하세요.

4

새 채팅 세션 활용

서로 다른 주제는 새로운 채팅으로 시작하여 컨텍스트를 명확히 유지하세요.

5

비피크 시간 사용

무료 플랜 사용자는 피크 시간을 피하면 더 빠른 응답을 받을 수 있습니다.

2025년 10월 현재, 주요 AI 서비스들은 무료 플랜으로도 유용한 기능을 제공하고 있습니다. 각 서비스의 특징과 제한사항을 이해하고 자신의 용도에 맞는 AI를 선택하면, 비용 부담 없이 최신 AI 기술을 효과적으로 활용할 수 있습니다.
